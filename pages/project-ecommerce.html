Project Overview

Architecture, deployment, and ongoing maintenance of a high-traffic B2B e-commerce platform designed for seamless retailer transactions at Samal Motors (a Tata Motors authorized distributor). The platform enables thousands of retailers to place orders, track real-time spare-parts inventory, and manage end-to-end distribution operations — all on AWS infrastructure engineered for 99.9% uptime.

Key Contributions

  • Traffic Management: Engineered scalable infrastructure using AWS Auto Scaling Groups and Application Load Balancers (ALB) to handle massive traffic surges during peak retailer ordering windows with zero performance degradation.
  • CI/CD Pipeline: Built and optimized automated Jenkins pipelines for continuous deployment of Spring Boot microservices, reducing manual deployment errors by 70%.
  • Infrastructure as Code: Provisioned and managed Dev, QA, and Production environments using Terraform — ensuring environment parity and rapid disaster recovery.
  • Container Orchestration: Deployed and managed microservices with Docker and Kubernetes; used Ansible for configuration management and streamlined update distribution.
  • Database & Caching: Administered Amazon RDS for order management and integrated Redis caching for real-time, low-latency inventory access.
  • Observability & Security: Implemented Prometheus + Grafana + CloudWatch monitoring stack; secured platform using AWS WAF and IAM to protect sensitive retailer data.
  • Automation Scripts: Authored Python and Bash scripts for daily DB backups, log rotations, and routine cloud maintenance tasks.

Architecture Notes

The platform follows a multi-tier architecture on AWS with public and private subnets. Web-facing traffic enters through an Application Load Balancer targeting Auto Scaling Groups of EC2 instances. Spring Boot microservices are containerized with Docker and orchestrated via Kubernetes. Amazon RDS handles persistent transactional data, with Redis providing a caching layer for inventory lookups. CloudWatch, Prometheus, and Grafana provide end-to-end observability across all tiers.
